# ReflexTester

A simple reflex testing game built with React and TypeScript. Test your reaction time by clicking when the screen turns green.

## Features

- Random delay (2-5 seconds) before the green signal
- Real-time reaction time measurement in milliseconds
- High score tracking (best reaction time)
- Visual feedback with color-coded states
- Click-too-early detection

## Getting Started

```bash
npm install
npm run dev
```

## How to Play

1. Click anywhere to start
2. Wait for the screen to turn green
3. Click as fast as possible when green appears
4. Try to beat your best time!

## Tech Stack

React • TypeScript • Vite • Tailwind CSS
