# Habit Tracker

A beautiful, minimalist habit tracking application built with React and TypeScript. Track your daily habits, maintain consistency, and build streaks with a clean, focused interface.

## Features

- **Add & Manage Habits** - Create and delete habits with ease
- **Daily Tracking** - Mark habits as complete for each day
- **Streak Counter** - Visual streak indicators with flame icons to track consecutive days
- **Local Storage** - All data persists in your browser's local storage
- **Modern UI** - Dark theme with emerald accents and smooth animations
- **Responsive Design** - Works beautifully on all screen sizes

## How to Run

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Usage

1. **Add a Habit**: Type a habit name in the input field and press Enter or click the plus button
2. **Mark Complete**: Click the checkmark button to toggle completion for today
3. **View Streaks**: See your current streak count displayed with a flame icon next to each habit
4. **Delete Habit**: Hover over a habit and click the trash icon to remove it

## Technical Details

- Built with **React 18** and **TypeScript**
- Uses **Vite** for fast development and building
- Styled with **Tailwind CSS**
- Icons from **Lucide React**
- Data persistence via **localStorage API**

## TypeScript Features Demonstrated

- Type-safe state management with TypeScript interfaces
- Proper typing for React hooks and event handlers
- Date manipulation with type safety
- UUID generation for unique identifiers
