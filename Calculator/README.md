# Calculator

A fully functional calculator application built with React and TypeScript, featuring keyboard support and a modern dark theme UI.

## Features

- Basic arithmetic operations (+, -, \*, /)
- Decimal number support
- Clear (AC) and backspace functionality
- Full keyboard support
- Modern dark theme UI

## How to Run

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Keyboard Shortcuts

- `0-9` - Number input
- `+`, `-`, `*`, `/` - Operators
- `Enter` or `=` - Calculate
- `Backspace` - Delete last digit
- `Escape` - Clear all

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
